## Python 指南

- `uv` 是 Python 包和环境管理的必备工​​具。

- 安装 Python 依赖项时，请使用 `uv` 和 `pyproject.toml` 文件，并确保安装到虚拟环境中。

- 运行 Python 工具时，尽可能使用 `uvx`。

- 如果需要添加新的依赖项，请说明包名及其添加的直接原因。例如：“添加 `httpx` 用于异步 API 调用。”

- 实现类型或类型提示时，请始终使用 PEP 585 中定义的 >=3.9 版本类型。使用内置类型（`dict`、`list`），而不是 `typing` 模块的等效类型。

- 为确保 Python 代码符合相关标准，在创建或修改任何 `.py` 文件之前，**必须**运行以下命令。从项目根目录运行以下命令：

```bash

uvx codespell@latest -s
# 检查拼写并显示摘要

uvx ruff@latest check --fix .
# 执行检查并修复

```
